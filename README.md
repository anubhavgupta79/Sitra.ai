# 🤖 Sitra.ai - The Ultimate AI-Powered Empowerment Suite

**Sitra.ai** ek advanced full-stack application hai jo artificial intelligence ko everyday tasks ke liye asaan banata hai. Isme image processing se lekar creative writing tak ke saare tools ek hi platform par available hain.

---

## 🚀 Key Features (AI Tools)

Sitra.ai mein aapko niche diye gaye saare powerful tools milte hain:

### 🎨 Image & Visual Tools
* **Background Removal:** Images se background ko instantly aur professionally remove karein.
* **Object Removal:** AI ki madad se kisi bhi unwanted object ko photo se saaf karein.
* **Image Generation:** Sirf text prompt likh kar high-quality, creative images banayein.

### ✍️ Content & Writing Suite
* **Blog Writer:** Kisi bhi topic par pura SEO-friendly blog likhein seconds mein.
* **Title Generator:** Apne videos ya articles ke liye catchy aur viral titles generate karein.

### 💼 Professional Career Tools
* **Resume Analyzer:** Apne resume ko analyze karwaein, score check karein aur ATS-friendly suggestions payein.

---

## 🛠️ Technology Stack

| Layer | Technology |
| :--- | :--- |
| **Frontend** | **React.js** (Fast & Responsive UI) |
| **Backend** | **Express.js & Node.js** (Robust Server-side Logic) |
| **AI Processing** | Google Gemini API / OpenAI / Stable Diffusion |
| **State Management** | React Hooks & Context API |
| **Styling** | Modern CSS / Tailwind CSS |

---

## 📂 Project Structure

Project do main parts mein divided hai:
* `client/`: Yahan saara Frontend (React) ka code hai.
* `server/`: Yahan Backend (Express) aur AI integration ka logic hai.

---

## ⚙️ Setup & Installation

1.  **Clone the Repo:**
    ```bash
    git clone [https://github.com/anubhavgupta79/Sitra.ai.git](https://github.com/anubhavgupta79/Sitra.ai.git)
    ```
2.  **Server Setup:**
    ```bash
    cd server
    npm install
    # .env file mein apni API_KEY daalein
    npm start
    ```
3.  **Client Setup:**
    ```bash
    cd client
    npm install
    npm start
    ```
